## Aby uruchomić program upewnij się, że masz zainstalowane środowisko Python oraz bibliotekę "pyserial".
#### Jeśli nie masz zainstalowanej biblioteki serial możesz to zrobić za pomocą komendy
```bash
pip install pyserial
```
### Następnie w terminalu wprowadź komendę:
```bash
python main.py
```
Teraz możesz skanować kody kreskowe.<br> 
Zeskanowane kody będą pojawiać się w terminalu
